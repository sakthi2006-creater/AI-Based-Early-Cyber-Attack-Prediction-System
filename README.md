# 📖 Project Overview

The AI-Based Early Cyber Attack Prediction System is designed to identify suspicious network activities before they become serious cyber attacks. The system uses machine learning algorithms to analyze network behavior, detect anomalies, and generate real-time alerts for security teams.

# 🚀 Installation

## Clone the Repository

```bash
git clone https://github.com/sakthi2006-creater/AI-Based-Early-Cyber-Attack-Prediction-System.git
```

## Move to Project Folder

```bash
cd AI-Based-Early-Cyber-Attack-Prediction-System
```

## Install Dependencies

```bash
npm install
```

## Start the Project

```bash
npm start
```

or

```bash
npm run dev
```

# ⚙️ How It Works

1. Collect network traffic data.
2. Preprocess the data.
3. Extract important features.
4. Apply Machine Learning models.
5. Detect suspicious activities.
6. Predict potential cyber attacks.
7. Display alerts on the dashboard.
